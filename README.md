本系统分为二种用户：

1.管理员：修改密码，用户管理，角色管理，菜单管理，新闻管理，留言管理，财务管理，社团管理，社团风采，社团活动，活动申请， 学生管理，社团申请

2.学生：登陆注册，查看社团，查看新闻，查看活动并申请参与，个人中心，申请社团等等

JAVA+tomcat+servlet+jsp+bootstrap
